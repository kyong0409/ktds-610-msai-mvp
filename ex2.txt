Azure AI 문서 처리 API 통합
Azure Document Intelligence로 문서 처리할 때, Python SDK부터 pip install azure-ai-documentintelligence --pre로 설치 (버전 맞춰야 함). 포털에서 리소스 그룹 만들어서 Document Intelligence 서비스 생성, 그 다음 API 키랑 엔드포인트 복사해서 os.environ["AZURE_DOCUMENT_INTELLIGENCE_ENDPOINT"] = "https://your-endpoint" 이런 식으로 환경 변수 설정. 코드로는 from azure.core.credentials import AzureKeyCredential; from azure.ai.documentintelligence import DocumentIntelligenceClient; client = DocumentIntelligenceClient(endpoint, AzureKeyCredential(key)). 분석은 client.begin_analyze_document("prebuilt-document", analyze_request=open("yourfile.pdf", "rb"), content_type="application/octet-stream") 호출, result = poller.result()로 기다림. PDF는 500페이지까지 되지만 무료 티어는 3페이지 제한, 이미지 추출은 result.pages에서 bounding boxes 봐야 하고, 테이블은 result.tables로 파싱. 에러 나면 401이면 키 확인, 429면 quota 초과. 가격은 페이지당 0.05달러쯤? JSON 결과는 custom model 훈련 안 하면 기본 필드만 나옴. 추가로 LangChain 연동 가능할 텐데, 그건 별도 (이전 검색 참조, Azure 관련).
